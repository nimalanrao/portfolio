# **App Name**: Nyxfolio

## Core Features:

- Profile Section: Displays the personal information including name, contact details, and a brief introduction.
- Skills Showcase: Highlights key skills in creative & marketing and technical areas, displayed in an organized manner.
- Project Gallery: Displays projects divided into content/photography and web/front-end categories with descriptions. Thumbnails link to detailed project pages.
- Service Offering: Lists services for brands including photography, content management, website building, and performance optimization.
- Tools Used: Lists all tools used in development, separated by use case.
- Contact Form: Includes contact details such as email, phone number, and links to Instagram, TikTok, and GitHub.
- Theme Toggle: Dynamically adjusts the color scheme between a light and dark theme based on the user's device preferences or manual selection.

## Style Guidelines:

- Background color: Off-white background (#F2F2F2) for light mode and dark gray (#222222) for dark mode.
- Primary color: Dark charcoal (#333333) for light mode text and light gray (#DDDDDD) for dark mode text.
- Accent color: A subdued teal (#77D8D8) for interactive elements and highlights, providing a touch of sophistication without overwhelming the minimalistic design.
- Body and headline font: 'Space Grotesk' for a modern, techy, and clean feel, suitable for both headlines and body text.
- Minimal, line-style icons used for skills and sections to maintain a clean and modern look.
- Clean, single-page layout with smooth scrolling to different sections.
- Subtle hover effects on portfolio items and call-to-action buttons to add interactivity without being distracting.
